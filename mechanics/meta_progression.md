# Meta Progression

## Overview
Solwara uses a meta progression system. Each run starts fresh but permanent unlocks accumulate over time.

## What Persists Between Runs
- Unlocked starters per role
- Move Mastery unlocks per role
- Charm pool expansions
- Legendary Charm (if purchased)
- Pokédex Charm per role (if unlocked)
- Story discoveries — things learned about the world stay learned
- The Hope Badge slot — always visible in the badge case, always a reminder
- Solwara Coins balance

## What Resets Each Run
- Your team
- Items
- In-run money (minus 10% converted to Solwara Coins on loss)
- Current charms equipped
- Node map progress

## Solwara Coins
- Persistent currency, never lost
- On a failed run: 10% of in-run money converts to Solwara Coins
- Encourages pushing further even in failing runs — more money earned = more coins
- Spent at the meta shop between runs

## Meta Shop
Accessible between runs. Purchased with Solwara Coins.

### Move Mastery
Role-exclusive moves teachable to starters. Moves the Pokémon could not normally learn in mainline.
Thematically tied to each roles personality — see move_mastery.md.

### Random Charm Unlock
Spend coins to permanently add a new charm to the in-run pool.
Random — you do not choose which charm is added.

### Legendary Charm
One time purchase. Adds legendaries to the rescue node pool.
Protected legendaries never available: Kyogre, Groudon, Dialga, Arceus.

### Pokédex Charm (per role)
Unlocked by beating a run with that role — not purchased with coins.
Allows selecting any previously caught Pokémon in base form as your starter for that role.
Role narrative still applies. Move Mastery still applies to chosen starter.
One charm per role — Hero, Doctor, Accepted, Defector (when available).

## Starter Unlock System
- Each role has a pool of 5 starters
- Default starter available immediately
- Beating a run unlocks a random starter from that roles pool
- Legendary starters have 10% unlock chance
- If only the legendary starter remains locked, next unlock is guaranteed
- Beating a role 4 times guarantees the legendary unlock path

## Charms (Relics)
Found during runs at loot nodes, shops, and events.
Passive effects that modify the run. Each run builds a different charm loadout.
New charms added to the pool permanently via meta shop purchases.
