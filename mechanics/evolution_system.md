# Evolution System

## Node Types

### Common Evolution Node
Handles friendship and miscellaneous evolutions that require special conditions in mainline.
Examples: Riolu → Lucario, Golbat → Crobat, Togepi → Togetic

### Uncommon Evolution Node
Handles item, trade, and Hisuian evolutions.
Also presents a regional form item — allows evolving into Alolan, Hisuian, Galarian, or Paldean forms where applicable.
Examples: Haunter → Gengar, Slowpoke → Slowking or Galarian Slowking, Stantler → Wyrdeer

### Rare Evolution Node
Allows early/illegal evolution of any level-based Pokémon before their natural level.
The exciting "do I take this now and lose future level-up moves?" decision.

## Rules
- Evolution nodes only spawn if a party member can use them — no dead nodes
- Player selects Pokémon → selects evolution path
- All available evolution options presented including regional variants
- Player can decline — another chance will come later in the run
- Level evolutions still happen automatically at set level as normal
- Galarian stage one lines are their own wild encounters, not regional evolution options

## Form Changes
All regional forms can be found in the wild as their own encounters.
Uncommon nodes handle Hisuian evolutions specifically as these require special conditions in mainline games.
