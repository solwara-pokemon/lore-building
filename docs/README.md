# Solwara — Lore & Design Bible

This repository contains all world-building, story, and design documentation for **Solwara**, a browser-based Pokémon fan game set in a flooded Oceania region.

## Structure

```
world/
  lore.md         ← The world, the flood, the canon basis
story/
  themes.md       ← Core narrative themes
  roles.md        ← Player roles and their arcs
  faction.md      ← The Unforgotten Party
  climax.md       ← The Kyogre revelation
pokemon/
  starters.md     ← All starter pools by role
mechanics/
  replayability.md ← Branching narrative, unlock systems
```

## Status
🟡 Active design phase — not yet in development.
