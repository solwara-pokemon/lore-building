# Move Mastery

## Overview
Move Mastery is the system by which starters learn moves they could not normally learn in mainline Pokémon. These moves are role-exclusive, thematically tied to each roles personality, and purchased from the meta shop using Solwara Coins.

Move Mastery applies to whichever Pokémon is your starter — including Pokédex Charm selections.

---

## The Hero
*Brave, selfless, protective. Carries the weight of others expectations.*

| Move | Thematic Reason |
|---|---|
| Sacred Sword | A heros blade — righteous, precise |
| Wish | Sacrifices your turn for someone else |
| Aura Sphere | Righteous energy, never misses |
| Reversal | Stronger when weakened — never gives up |
| Close Combat | Throws yourself in regardless of consequence |

---

## The Doctor
*Curious, driven, interventionist. Always moving, always solving.*

| Move | Thematic Reason |
|---|---|

| Future Sight | Sees what is coming before it arrives |
| Psych Up | Adapts to whatever it faces |
| Skill Swap | Always has a plan involving switching things around |
| Miracle Eye | Sees through barriers, finds what others miss |
