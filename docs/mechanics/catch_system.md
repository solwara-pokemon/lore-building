# Catch System

## Acquisition Methods

### Wild Nodes
Standard capture — weaken Pokémon and throw a ball. Familiar mainline mechanic.

### Rescue Nodes
Free Pokémon, no battle, no cost. Player simply chooses to take it or leave it.

**Rescue node spawn pool:**
- Any Pokémon in the game
- Rarity based on BST — lower BST = more common, higher BST = more rare
- Form changes do not factor into BST calculation (e.g. Wishiwashi uses base form BST)
- Legendaries excluded by default
- Legendaries added to pool when Legendary Charm is purchased from meta shop
- Protected legendaries never available even with Legendary Charm: Kyogre, Groudon, Dialga, Arceus

### Shops
Strong Pokémon available for purchase. Buying a Pokémon means forgoing any item from that shop — a real tradeoff.

### Trainer Battles
No catching from trainer battles. Period.

## Trainer Catching
Not permitted under any circumstances.
