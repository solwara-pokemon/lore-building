# Starter Pokémon

## System Overview
- Each role has a pool of 5 starters
- The default starter is available from the beginning
- Beating a run unlocks a random starter from that roles pool
- Legendary starters have a 10% unlock chance; guaranteed if all others are unlocked
- Unlockable starters may bend 1-2 mainline evolution rules for thematic fit
- **No fakemon.** All starters are existing Pokémon.
- The Fire/Water/Grass triangle is not required.
